E.V.A. GEAR

This project was started in late 2018 as a hobby. It was aimed to be a real robot 3rd person action game on PC with the anime art style. It was inspired by many games and animes such as Gundam, Ace Combat, Metal Slug and Front Mission. Because of the art style and budget, Unity and Blender 3D were the suitable choice for this project also it is a good opportunity to learn Unity's HDRP(High Definition Render Pipe Line) and 3d modelling at the same time.

PHASE 1

This is the very first phase of the project. The primary objective of this phase is essential functions such as movement, animation, target lock-on, HUD, post-processing effect, etc. The biggest success in this phase is accurate targeting system combine with 3-joints inverse kinematic system that can achieve nearly pinpoint accuracy. (for simple linear movement target) full video link here...

PHASE 2

The second phase is about the character's model and environment. Unity's shader graph is used to make the shader for coconut's leaves and wave ripples. the sea wave is the combination of 2 sine waves with different direction, wavelength and amplitude. The coconut's leaves are animated by gradient noise weighted with vertex position and the trees were planted by code with random deviation in position, rotation, and scale. full video link here...
